## 1. 增量概述

### 1.1 迭代目标

1. **会员功能**：设计并实现完整的会员等级体系、额度系统、支付订阅流程，实现商业变现闭环
2. **功能完善**：补全P1核心功能、精选P2功能，提升产品完整度与用户体验

### 1.2 迭代范围

| 维度 | 包含 | 不包含 |

- 会员转化率：Free → Pro 转化引导触达率 > 80%
- 功能覆盖：P1 功能 100% 上线，P2 精选功能 1 项上线

### 2.2 额度系统设计

#### 2.2.1 额度类型

| 类型 | 说明 | 重置规则 |
|------|------|----------|
| 日额度（Daily Quota） | 每日可用的生成次数，分图片/视频独立计算 | 每日 00:00 重置 |
| 积分（Credits） | 可跨日累积的虚拟货币，1积分 = 1次图片生成，3积分 = 1次视频生成 | 不自动重置，可购买 |
| 功能额度 | 特定功能的可用次数（如批量生成次数） | 与日额度同步重置 |

#### 2.2.2 额度消耗规则

| 操作 | Free | Pro | Enterprise |
|------|------|-----|------------|
| 文生图（单张） | 1次日额度 | 1次日额度 | 不计额度 |
| 文生视频 | 1次日额度 | 1次日额度 | 不计额度 |
| 图生图（单张） | —（不可用） | 1次日额度 | 不计额度 |
| 图生视频 | —（不可用） | 2次日额度 | 不计额度 |
| 批量生成2张 | —（不可用） | 2次日额度 | 不计额度 |
| 批量生成4张 | —（不可用） | 4次日额度 | 不计额度 |

#### 2.2.3 积分购买方案（Mock）

| 包名 | 积分数量 | 价格 | 有效期 |
|------|----------|------|--------|
| 体验包 | 10积分 | ¥9.9 | 30天 |
| 标准包 | 50积分 | ¥39 | 90天 |
| 畅享包 | 200积分 | ¥129 | 180天 |

#### 2.2.4 额度状态数据结构

```typescript
interface QuotaState {
  daily: {
    image: { used: number; total: number; resetAt: string }; // ISO datetime
    video: { used: number; total: number; resetAt: string };
  };
  credits: {
    balance: number;
    expiringAt?: string; // 最近过期时间
  };
  tier: 'free' | 'pro' | 'enterprise';
  subscription?: {
    plan: 'monthly' | 'yearly';
    startedAt: string;
    expiresAt: string;
    autoRenew: boolean;
  };
}
```

### 2.3 支付与订阅流程（Mock）

#### 2.3.1 订阅流程

```
会员中心 → 选择等级(Pro/Enterprise) → 选择计费周期(月付/年付) → 确认订单 → Mock支付 → 开通权益
```

**Mock支付规则**：
- 点击"确认支付"后模拟1.5秒加载动画
- 100%支付成功（Mock阶段不模拟失败）
- 支付成功后立即更新会员等级和额度
- 弹出支付成功提示，引导返回创作台

#### 2.3.2 积分购买流程

```
会员中心 → 积分商城 → 选择包名 → 确认购买 → Mock支付 → 积分到账
```

#### 2.3.3 升级/降级规则

| 场景 | 规则 |
|------|------|
| Free → Pro | 立即生效，当日额度按Pro标准刷新 |
| Pro → Enterprise | 立即生效，补差价（Mock阶段直接切换） |
| Pro → Free（到期） | 到期日24:00降级，保留历史数据，额度恢复Free标准 |
| 降级后收藏超限 | 保留现有收藏，但不可新增直到降至限额内 |

## 3. P1功能完善

### 3.1 P1-01 图生图

**前置条件**: Pro 及以上会员

**交互流程**:
1. 创作台切换到"图生图"Tab
2. 上传参考图片（支持拖拽/点击上传，JPG/PNG/WebP，≤10MB）
3. 参考图以缩略图显示在左侧，可替换/删除
4. 输入文字描述（描述期望的变化方向）
5. 选择风格（可选，基于参考图风格推荐）
6. 调节"参考强度"滑块（0%~100%，控制与原图的相似程度）
7. 点击生成 → 消耗1次日额度
8. 结果展示：并排显示参考图与生成图，方便对比

**Free用户引导**: 显示功能卡片但置灰，点击弹出"升级Pro解锁图生图"引导弹窗

**组件依赖**: ImageUpload、StyleSelector、ReferenceSlider、ResultCompare

### 3.2 P1-02 图生视频

**前置条件**: Pro 及以上会员

**交互流程**:
1. 创作台切换到"图生视频"Tab
2. 上传静态图片（JPG/PNG/WebP，≤10MB）
3. 预览图片，可添加运动描述（如"镜头向右平移"、"头发飘动"）
4. 选择视频时长（3秒/5秒）
5. 选择分辨率（Pro: 720P/1080P）
6. 点击生成 → 消耗2次日额度
7. 生成结果以视频播放器展示，支持播放/暂停/全屏

**Free用户引导**: 同P1-01，置灰+升级引导

**组件依赖**: ImageUpload、MotionInput、DurationSelector、VideoPlayer

### 3.3 P1-03 Prompt模板

**模板分类**:

| 分类 | 模板数 | 示例 |
|------|--------|------|
| 人物肖像 | 5 | "职业照 - 穿西装的商务人士，专业灯光" |
| 风景场景 | 5 | "日落海滩 - 金色阳光，椰子树剪影" |
| 创意艺术 | 5 | "赛博朋克城市 - 霓虹灯，未来建筑" |
| 产品展示 | 3 | "极简产品图 - 白色背景，柔和阴影" |
| 动物自然 | 3 | "微距蝴蝶 - 花瓣上，散景背景" |
| 抽象图形 | 3 | "流体艺术 - 渐变色彩，有机形态" |

**访问控制**:
- Free: 基础模板（人物肖像/风景场景/创意艺术各1个，共3个）
- Pro: 全部24个模板 + 自定义保存（最多20个）
- Enterprise: 全部模板 + 不限自定义 + 团队共享模板

**交互设计**:
1. 创作台Prompt输入框下方显示"✨ Prompt模板"按钮
2. 点击展开模板库侧边抽屉（Drawer）
3. 左侧分类Tab，右侧模板卡片网格
4. 模板卡片展示：缩略图预览 + 模板名称 + Prompt摘要
5. 点击模板 → Prompt自动填入输入框（可编辑修改）
6. Pro用户可将当前Prompt保存为自定义模板

**组件依赖**: PromptTemplateDrawer、TemplateCard、CustomTemplateSaveDialog

### 3.4 P1-04 生成参数调节

**前置条件**: Pro 及以上会员可调节高级参数

**参数面板设计**:

| 参数 | Free | Pro | Enterprise | 范围 | 默认值 | 说明 |
|------|------|-----|------------|------|--------|------|
| 创意度（CFG Scale） | ❌ | ✅ | ✅ | 1-30 | 7 | 值越高越贴合Prompt |
| 生成步数（Steps） | ❌ | ✅ | ✅ | 10-50 | 20 | 步数越多质量越高，耗时越长 |
| 参考强度 | ❌ | ✅ | ✅ | 0-100% | 50% | 仅图生图模式可用 |
| 种子值（Seed） | ❌ | ❌ | ✅ | 整数 | 随机 | 固定种子可复现结果 |
| 负面Prompt | ❌ | ✅ | ✅ | 文本 | 空 | 排除不想要的元素 |

**交互设计**:
1. 创作台"高级设置"折叠面板（默认收起）
2. 展开后显示可用参数（根据会员等级动态显示）
3. 每个参数：标签 + 滑块/输入框 + 当前值 + 重置按钮
4. Free用户看到参数项但标注"Pro"，点击弹出升级引导

**组件依赖**: AdvancedSettingsPanel、ParameterSlider、UpgradeHint

### 3.5 P1-05 收藏与标签增强

**与会员体系联动**:

| 功能 | Free | Pro | Enterprise |
|------|------|-----|------------|
| 收藏数量 | 最多10个 | 不限 | 不限 |
| 自定义标签 | ❌ | ✅ 创建/编辑/删除 | ✅ + 团队共享标签 |
| 标签筛选 | 仅系统标签 | 系统+自定义标签 | 全部标签 |
| 批量操作 | ❌ | 批量收藏/删除 | 批量 + 团队批量 |

**交互增强**:
1. 收藏按钮旁显示当前收藏数/上限（Free: "3/10"）
2. 接近上限时（Free ≥ 8个），收藏时提示"即将达到收藏上限，升级Pro解锁无限收藏"
3. 达到上限后，收藏按钮变为"升级Pro"引导按钮
4. 标签管理：Pro用户可在历史页"标签管理"弹窗中CRUD自定义标签
5. 自定义标签以不同颜色区分于系统标签

**组件依赖**: FavoriteButton、TagManager、TagChip、UpgradeBadge

### 3.6 P1-06 批量生成

**前置条件**: Pro 及以上会员

**功能设计**:

| 会员等级 | 批量选项 | 单次上限 |
|----------|----------|----------|
| Free | 不支持（仅单张） | 1 |
| Pro | 2张/4张 | 4 |
| Enterprise | 2张/4张/8张 | 8 |

**交互设计**:
1. 创作台生成按钮旁显示"批量生成"切换（仅Pro+可见）
2. 点击切换后弹出批量选项：2张/4张/（8张）
3. 选择后按钮文案变为"生成 2 张"、"生成 4 张"
4. 生成过程：展示进度条（1/4 完成...）
5. 结果展示：网格布局展示所有生成结果，支持逐个查看/下载/收藏
6. 额度消耗：按实际生成张数扣除日额度

**Free用户引导**: 生成按钮旁显示"批量生成🔒"，点击弹出升级引导

**组件依赖**: BatchGenerateToggle、BatchProgress、ResultGrid

## 4. P2功能精选

### 4.1 P2-02 社区分享（简化版）

**功能范围**:
- ✅ 分享自己的作品到社区
- ✅ 浏览作品广场（瀑布流）
- ✅ 点赞
- ✅ 按分类/热门/最新筛选
- ❌ 评论系统（后续迭代）
- ❌ 关注用户（后续迭代）
- ❌ 举报/审核（Mock阶段暂不需要）

**会员差异**:

| 功能 | Free | Pro | Enterprise |
|------|------|-----|------------|
| 浏览广场 | ✅ | ✅ | ✅ |
| 分享作品 | ✅ | ✅ | ✅ + 置顶 |
| 每日分享次数 | 3次 | 不限 | 不限 |
| 作品置顶 | ❌ | ❌ | ✅（3个/月） |

**交互设计**:

**分享流程**:
1. 生成结果页/历史记录中点击"分享到社区"按钮
2. 弹出分享编辑弹窗：
   - 选择要分享的图片/视频（批量生成时可选其中一张）
   - 添加分享描述（≤200字，可选）
   - 选择分类标签（从预设列表选择1-3个）
   - 勾选"公开分享Prompt"（默认不公开）
3. 确认分享 → 扣除1次分享额度

**作品广场**:
1. 底部导航栏增加"广场"Tab（图标: explore/public）
2. 顶部筛选栏：最新 / 最热 / 分类下拉
3. 瀑布流布局，卡片展示：缩略图 + 标题 + 点赞数 + 作者头像
4. 点击卡片 → 全屏预览 + Prompt（如公开）+ 点赞按钮
5. 双击卡片快速点赞（带心形动画）

**组件依赖**: ShareDialog、GalleryPage、WaterfallLayout、GalleryCard、LikeButton
